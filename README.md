# Projekt_PSI_Sentyment_Piosenki
📥 Instrukcja przygotowania danych (Dane wejściowe)
Wszystkie dane tekstowe muszą znajdować się w folderze data/ i być zakodowane w formacie UTF-8. System wykorzystuje ustrukturyzowany nagłówek na początku pliku tekstowego do ekstrakcji metadanych. Metadane pobieramy z Wikipedii. Rok podejmy, kiedy piosenka znajdowała się na lisćie Billboard Year-End Hot 100 singles (https://en.wikipedia.org/wiki/Category:Lists_of_Billboard_Year-End_Hot_100_singles). Teskt piosenki kopiujemy z Genius (https://genius.com/).
